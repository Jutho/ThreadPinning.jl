References - OpenBLAS

ThreadPinning.openblas_getcpuid — Function

Returns the ID of the CPU thread on which the i-th OpenBLAS thread is currently running.

Note

Available as of Julia 1.9.

source
ThreadPinning.openblas_getcpuids — Function

Returns the IDs of the CPU threads on which the OpenBLAS threads are currently running.

Note

Available as of Julia 1.9.

source

Pin the available OpenBLAS threads to the given CPU ids. Currently, only explict pinning is possible.

source
